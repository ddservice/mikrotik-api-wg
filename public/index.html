Export CSV (พรบ)
ระบบบันทึกโดเมนที่เข้าชมอัตโนมัติทุก 5 นาที ตาม พรบ ว่าด้วยการกระทำความผิดเกี่ยวกับคอมพิวเตอร์ พ.ศ.2560 มาตรา 26 — เก็บรักษา 90 วัน

Username/Password คือบัญชี login เข้า RouterOS API ของราวเตอร์ตัวนี้ (คนละเรื่องกับ WireGuard) — ต้องกรอกตั้งแต่ครั้งแรก ไม่งั้น dashboard จะเชื่อมต่อราวเตอร์ไม่ได้เลยไม่ว่าจะใช้ WireGuard หรือ Direct IP ก็ตาม แก้ไขทีหลังได้เสมอ (ช่อง Password ตอนแก้ไข ปล่อยว่างไว้ได้ถ้าไม่ต้องการเปลี่ยนรหัสผ่านเดิม)
ขั้นตอนแนะนำสำหรับ WireGuard: (1) เลือก Connection Type = WireGuard และกรอก WireGuard IP + Port ด้านล่างนี้ก่อน (2) กดปุ่ม "สร้างสคริปต์ WireGuard" ท้ายฟอร์ม (จะดึงค่าที่กรอกไว้นี้ไปใช้อัตโนมัติ) (3) รันสคริปต์บนราวเตอร์ (4) ค่อยกลับมากด "บันทึกไซต์งาน" ที่นี่อีกครั้ง
ต้องไม่ซ้ำกับไซต์งานอื่นที่มีอยู่แล้ว
ถ้าใช้ WireGuard ควรตรงกับ "WireGuard IP" ด้านบน — ระบบจะเติมให้อัตโนมัติเมื่อเลือก Connection Type เป็น WireGuard
